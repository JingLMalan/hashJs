# hashJs

[![Greenkeeper badge](https://badges.greenkeeper.io/JingLMalan/hashJs.svg)](https://greenkeeper.io/)

hashTable 

**开箱即用的Hash算法

**应用： 1.通讯录
      2. DNS解析
      3. 防止重复
      4. 缓存
  
**用法
   1. 使用yarn安装依赖: yarn install || yarn 
   2. 使用babel编译: yarn build
   3. 运行hash算法： node dist/index.js
